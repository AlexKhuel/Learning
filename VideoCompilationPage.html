How to Code: Basic HTML Elements

HTML is not a coding language in the traditional sense. You would not use it to do math or make a game, but it is still an essential part of programming. HTML is a kind of "translator" between code and humans that allows things to be shown on the web.

Every good website has a button. Whether its to take you to another page, submit a form, or reveal something, the button is an essential part of your toolkit. Learn how to code it below.

How to code sidebar menus

The sidebar menu is the best way to help people navigate your site. You can have links to pages that can be categorized in subfolders. Learn how to code it below.
